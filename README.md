# Smart Horizon Hackathon - Participant Registration

This website is dedicated to participant registration for the Smart Horizon 48-Hour International Hackathon 2026 organized by New Horizon College of Engineering.

## Features

- **Participant Registration**: Complete team registration form for hackathon participants
- **Responsive Design**: Mobile-friendly interface
- **Form Validation**: Client-side validation for all form fields
- **Team Management**: Support for teams of 4-5 members

## Files Structure

- `index.html` - Main landing page with hackathon information
- `participant_registration.html` - Team registration form
- `participant_script.js` - JavaScript for form validation and functionality
- `participant_style.css` - Styles for registration form
- `style.css` - Main website styles
- `script.js` - Main website JavaScript
- `participant_database_setup.sql` - Database schema for participant data

## Usage

1. Open `index.html` in a web browser to view the main website
2. Navigate to participant registration to register your team
3. Fill out all required team member information
4. Submit the form to complete registration
